# Aroma — лендинг кафе

Одностраничный сайт кафе «Aroma» с разделами «Главная», «Меню», «О нас», «Отзывы» и «Контакты» с формой обратной связи.

## Структура
- `index.html` — основная страница с разметкой всех разделов.
- `assets/css/style.css` — стилизация, адаптивная навигация и оформления карточек/формы.
- `assets/js/script.js` — взаимодействие: мобильное меню и уведомление при отправке формы.

## Просмотр
Откройте `index.html` в браузере или поднимите любой локальный сервер, например:

```bash
python -m http.server 8000
```

После запуска перейдите в браузере на `http://localhost:8000`.
